SoulWorker Skill Simulator

The skill simulator is made based on the information from the Korean & Steam (Global) version of SoulWorker.

This skill simulator require javascript turned on.
Javascript is disabled.
The skill simulator should work on most of modern browsers. Tested on (02.January.2021): Firefox v84, Chrome v87, Edge v87, Opera v72.

As of now, the simulator video has skill preview video (which require Promise object).
